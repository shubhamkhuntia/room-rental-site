# Furnished Room Rental Website

Static listing website for a furnished private room in Sai Enclave, Balianta, Hanspal, Bhubaneshwar 752101.

## Deploy

This is a plain HTML/CSS site. On Vercel:

1. Import this Git repository.
2. Use the default static site settings.
3. Leave build command empty.
4. Leave output directory empty or set it to `.`.
